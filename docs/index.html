Capabilities

Everything your system needs

From raw sensor readings to fully autonomous nutrient management — all in one integrated platform.

🌡

Real-Time Sensor Monitoring

Live readings of pH, EC, water temperature, reservoir level, air temperature, humidity, and VPD — streamed directly to your dashboard via WebSocket.

🧪

Autonomous pH & EC Dosing

PID-controlled dosing pumps precisely deliver pH-up, pH-down, and nutrient solutions to maintain your configured target values within tight tolerances.

⚙️

Configurable Automation Rules

Set target values, tolerance bands, dose limits, and PID tuning constants via the dashboard UI with instant backend persistence — no config files to edit.

📡

ESP32 MQTT Firmware

Custom PlatformIO firmware running on ESP32 communicates over MQTT. Supports solenoid valve control, dosing pump PWM, and multi-sensor polling.

📊

Dosing History & Logs

Every automatic and manual dose is recorded with timestamps, pump index, and duration so you can audit and tune the system over time.

📲

Progressive Web App

Install HydroMonitor on your phone or tablet like a native app. Works on any device — no app store required. Builds on React with Vite for instant load times.
